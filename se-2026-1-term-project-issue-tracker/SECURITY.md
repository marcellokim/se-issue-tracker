# 보안 정책

이 저장소는 GitHub의 보안 기능을 활용해 취약점을 관리합니다.

## 취약점 신고 방법
민감한 보안 이슈는 **공개 Issue로 올리지 마세요.**

가능하면 아래 순서로 신고해주세요.

1. GitHub 저장소의 **Security** 탭으로 이동
2. **Report a vulnerability** 기능 사용
3. 재현 방법, 영향 범위, 예상 원인, 완화 방법을 최대한 구체적으로 작성

## 현재 활성화된 보안 기능
- Dependabot 보안 업데이트
- Secret scanning
- Secret scanning push protection
- Private vulnerability reporting
- GitHub code scanning 기본 설정

## 권장 신고 내용
- 취약점 유형
- 재현 절차
- 영향 범위
- 악용 가능성
- 임시 완화책

## 공개 이슈로 작성해도 되는 경우
- 민감 정보 노출이 없는 일반 보안 개선 요청
- 이미 공개된 의존성 업데이트 제안
- 취약점 자체가 아닌 정책/문서 개선 요청
