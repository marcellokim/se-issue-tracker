SD 09 Close Issue - GRASP 적용 설명

관련 산출물:
- UC6 Change Issue State
- SSD 09: changeStatus(issueId, targetStatus=CLOSED, comment)
- OC-08: status=CLOSED, assignee/verifier 제거, fixer/resolver 보존

적용 GRASP:

1. GRASP Controller
`:IssueTrackingSystem` 객체가 PL의 close 요청을 받는다. 여기서 IssueTrackingSystem은 MVC 클래스가 아니라 상태 변경 유스케이스를 조정하는 GRASP Controller 역할의 설계 객체이다.

2. Information Expert
Issue는 status, assignee, verifier, fixer, resolver를 알고 있으므로 close 처리 책임을 가진다.

3. Creator
close comment와 COMMENTED/STATUS_CHANGED history는 해당 Issue의 기록이므로 Issue가 생성한다.

4. Low Coupling / High Cohesion
GRASP Controller가 assignee/verifier를 직접 제거하지 않고 Issue.close(...)에 의도를 전달한다.
